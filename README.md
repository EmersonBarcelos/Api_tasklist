# Sobre
API de lista de tarefas, com autenticação de usuários e criptografia de senhas, junto com conexão a um banco de dados PostGreSQL

<h2>Tecnologias</h2>
 
  <li>
    <ul>bcrypt</ul>
    <ul>cors</ul>
    <ul>express</ul>
    <ul>jsonwebtoken</ul>
    <ul>pg-hstore</ul>
    <ul>sequelize</ul>
    <ul>yup</ul>
  </li>
